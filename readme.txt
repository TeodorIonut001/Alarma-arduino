Proiectul este conceput sa functioneze cu ajutorul unui arduino, senzor de inclinare (giroscopic) si unui senzor RFID.
Acesta percepe miscarile subtile pe care le face autovehiculul (in exemplul de fata, este programat sa functioneze pentru motociclete)
si porneste o alarma. Se armeaza / dezarmeaza / opreste alarma la contactul unei cartele cu senzorul RFID.
